# ER-case-communication.skill
覆盖内容  5 大场景全覆盖：违纪解除、协商解除 / 劝退、绩效改进 (PIP)、调岗调薪、员工投诉 —— 每个场景都有完整的分步操作流程。  结合败诉案例的实战经验：基于 24-26 年实际败诉数据，提炼了四大高风险模式：   处罚过重（33%）→ 内置 "处罚适配评估矩阵"，8 个维度逐项打分  证据链断裂（25%）→ 内置证据保全 SOP，按证据类型明确保全要求  程序瑕疵（17%）→ 内置程序合规核查清单，逐项打勾  管理漏洞（25%）→ 明确区分真实违纪 vs 借名处理   三步合规底线：合法（法律依据）→ 合理（处罚适配）→ 合情（沟通温度），每个环节都有对应的话术模板和检查清单。  适用人群：后台 HR、前台 HRBP、各级管理者，都能按图索骥使用。
